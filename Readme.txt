＝＝＝＝＝＝＝追加アイテム＝＝＝＝＝＝＝
・*Ore Searcher*
オフハンドに持つか、ホットバーで手に持つことで
自身の視線の先に何の鉱石が埋まっているかを探知するアイテムです。

手に持っている状態で右クリックをすることで、どの鉱石のみを探知するか選択することが出来ます。

最大射程は15mです。
1～6mまでは自身の視線そのままの探知しか行いませんが、7m以後は若干探知範囲が広くなります。

＝＝＝＝＝＝＝レシピ＝＝＝＝＝＝＝＝＝＝
・*Ore Searcher*
-作成方法-
*B.E.P Fairy Tech Crafter*に、以下の通りにアイテムを一つづつ入れることで作成されます。
X:空欄
A:スカルクセンサー
B:調律されたスカルクセンサー
C:ネザライトインゴット
D:ダイヤモンドブロック
E:スカルクシュリーカー
